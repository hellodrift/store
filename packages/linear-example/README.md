# Linear

Linear issue tracking integration for Drift.

## Features

- Create, edit, and manage issues
- Transition issues between workflow states
- Assign users, add/remove labels, and manage comments
- Search issues with full-text search
- Discover teams, members, labels, projects, and cycles
- Archive and unarchive issues
- Create and manage issue relations (blocks, duplicates, related)

## Authentication

Connect via OAuth (recommended) or paste a Linear API key in Settings.
