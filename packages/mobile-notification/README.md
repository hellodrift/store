# Mobile Notifications

Send push notifications to your own mobile devices via Drift.

## Features

- Send push notifications with custom title, body, and category
- Notifications are delivered to all your registered mobile devices
- Supports notification categories: messages, mentions, updates, marketing, automations

## Authentication

Automatically uses your Drift account credentials. No additional setup required.
