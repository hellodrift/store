# New Skill to Build Screens

We're excited to announce the launch of a brand new skill designed to make building custom screens and UI components easier than ever. This skill simplifies the process of creating, testing, and deploying custom interfaces within Drift.

## What's New

- **Screen Builder**: Intuitive tools for designing custom layouts and components
- **Live Preview**: See your changes in real-time as you build
- **Component Library**: Access pre-built components to accelerate development
- **Easy Integration**: Seamlessly integrate your custom screens into your Drift workflows

## Key Features

- **Drag-and-drop interface** — No need to write complex code from scratch
- **React-based components** — Leverage React for powerful, interactive UIs
- **Built-in testing** — Test your screens before deploying to production
- **Hot reload** — Instantly see updates without restarting

## Get Started

The new screen-building skill is available now. Check out the documentation to learn how to create your first custom screen. Visit the Drift skills marketplace to explore examples and templates.

Try it today and unlock the potential to build tailored experiences for your team!
